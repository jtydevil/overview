# 项目初始化
````
# 指定路径，保证最新安装工具
composer global config repositories.lumen-installer git https://github.com/YunhanPHP/lumen-installer.git

# 下载安装工具
composer global require yunhanphp/lumen-installer

# 初始化项目
lumen-app new projectName --inner
````
